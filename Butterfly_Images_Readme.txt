Welcome to the Butterfly Images
======================================

These images are intended to be used as examples for the Butterfly Viewer and Butterfly Registrator.
The images are created and owned by the Van Gogh Museum, but licensed under Creative Commons Attribution-NonCommercial 4.0 International. 
To view a copy of this license, visit https://creativecommons.org/licenses/by-nc/4.0/.
Additional metadata (including a credit line) for the images is recorded in a separate PDF file included in this folder.

The latest version of the Butterfly Viewer itself is available at https://github.com/olive-groves/butterfly_viewer.
